│   ├── esbuild.config.mjs      # Build config
│   └── .gitignore              # Git ignore rules
│
├── 📁 src/                      # Source code

│
├── 📁 docs/                     # Documentation

## File Descriptions

### Root Level
- **main.js** - Compiled plugin bundle (DO NOT EDIT - auto-generated)
- **manifest.json** - Plugin metadata for Obsidian
- **styles.css** - CSS styling for UI
- **package.json** - NPM dependencies and build scripts
- **tsconfig.json** - TypeScript compilation settings
- **esbuild.config.mjs** - Bundler configuration

### src/ (Source Code)
- **main.ts** - Main TypeScript source code
  - SyncModal class (UI)
  - MemosSyncPlugin class (core logic)
  - Helper functions

### docs/ (Documentation)
- **README.md** - Complete user guide
- **QUICKSTART.md** - 5-minute setup guide
- **TROUBLESHOOTING.md** - Problem-solving guide
- **CHANGELOG.md** - Version history and roadmap

### scripts/ (Installation)
- **install.ps1** - PowerShell installer for Windows
- **install.sh** - Bash installer for macOS/Linux

## Development Workflow

### Build Process
```bash
npm install          # Install dependencies once
npm run dev         # Build in watch mode (auto-rebuild on changes)
npm run build       # Production build
```

### File Flow
```
src/main.ts
    ↓ (TypeScript compiler)
    ↓ (esbuild bundler)
    ↓
main.js (compiled output)
    ↓ (copy to Obsidian)
    ↓
~/.obsidian/plugins/obsidian-memos-sync/main.js
```

## Git Configuration

### Tracked Files
✅ Committed to git:
- `src/main.ts` - Source code
- `manifest.json` - Metadata
- `styles.css` - Styling
- `docs/` - Documentation
- `scripts/` - Install scripts
- `*.json` (config files) - Configuration

### Ignored Files
❌ NOT committed (in .gitignore):
- `main.js` - Generated (can be rebuilt)
- `node_modules/` - Dependencies (can be reinstalled)
- `.vscode/`, `.idea/` - IDE settings
- `.DS_Store`, `Thumbs.db` - OS files
- `*.tmp` - Temporary files

## Build Output

After running `npm run build`:
- `main.js` is created in root
- This is the compiled plugin bundle
- Ready for distribution

## Installation Distribution

For users:
```
1. Run: scripts/install.ps1 (Windows)
           or
           scripts/install.sh (macOS/Linux)
   
2. This copies to: ~/.obsidian/plugins/obsidian-memos-sync/
   - main.js
   - manifest.json
   - styles.css
```

## Editing Guidelines

### ✅ DO Edit
- `src/main.ts` - Plugin source code
- `styles.css` - Styling changes
- `docs/*.md` - Documentation
- `package.json` - Dependencies only

### ❌ DON'T Edit
- `main.js` - Rebuild instead (npm run build)
- `package-lock.json` - Auto-managed
- `.gitignore` - Standard for all plugins
- `manifest.json` - Unless version bump needed
